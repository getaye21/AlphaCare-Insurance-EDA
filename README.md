# AlphaCare Insurance Ethiopia – EDA

## Objective
Identify low‑risk customer segments (by postal code, legal type, gender) to reduce premiums and attract new clients.

## Data
50 rows of synthetic Ethiopian car insurance data (2019–2024).  
Columns: Premium, Claims, DriverAge, VehicleAge, PostalCode, etc.

## How to run
1. Clone this repo.
2. Install: `pip install pandas matplotlib seaborn`
3. Run: `python eda.py`
4. See results in `outputs/` folder.

## Results
- Low‑risk segments saved in `outputs/low_risk_segments.csv`
- Charts: premium distribution, loss ratio by postal code, etc.
